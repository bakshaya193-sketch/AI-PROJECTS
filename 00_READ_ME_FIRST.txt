================================================================================
                              🆘 COMMON ISSUES
================================================================================

"Python not found"
→ Install Python from python.org and add to PATH

"venv not working"
→ Delete venv folder and recreate it

"ModuleNotFoundError"
→ Make sure venv is activated (see (venv) prefix)
→ Run: pip install -r requirements.txt again

"Port 8000/5173 in use"
→ Another app is using the port
→ Close other terminal windows or kill the process

"No response from AI"
→ Check OpenAI key in .env file is correct
→ Upload sample documents first

See README.md TROUBLESHOOTING section for more help
